Instrucciones para ejecutar el código - ENTRENAMIENTO Y EVALUACIÓN UNDERSAMPLING:

1. Abrir la carpeta del proyecto:
   - Navega a la siguiente dirección en el CD:
     ANÁLISIS COMPARATIVO DE ALGORITMOS DE MACHINE LEARNING SUPERVISADO PARA LA CLASIFICACIÓN DE FALLAS DE MAQUINARIA INDUSTRIAL\5.ENTRENAMIENTO Y EVALUACIÓN UNDERSAMPLING

2. Abrir Visual Studio Code:
   - Lanza Visual Studio Code.

3. Abrir la carpeta en VS Code:
   - Haz clic en Archivo > Abrir Carpeta...
   - Selecciona la carpeta "5.ENTRENAMIENTO Y EVALUACIÓN UNDERSAMPLING" que contiene el código.

4. Verificar el archivo de preparación de datos:
   - Asegúrate de que el archivo "PREPARACION_DATOS.csv" esté en la misma carpeta o en la ruta correcta según el código.

5. Ejecutar el código:
   - Abre el archivo de código (Jupyter Notebook).
   - Ejecuta el código, el cual leerá el archivo "PREPARACION_DATOS.csv" y realizará el entrenamiento y evaluación con la técnica de UnderSampling.
